# Bootstrap Structure Notes

When Bootstrap is downloaded locally, the main folders are:

- `css`: compiled Bootstrap stylesheets such as `bootstrap.min.css`
- `js`: JavaScript plugins and `bootstrap.bundle.min.js`
- `icons`: Bootstrap Icons assets when the icon package is installed separately

The sample page uses CDN links for easy browser execution. A downloaded setup can replace the CDN paths with local `css/bootstrap.min.css`, `js/bootstrap.bundle.min.js`, and Bootstrap Icons paths.
