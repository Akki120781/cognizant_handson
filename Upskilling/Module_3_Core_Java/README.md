# Module 3: Core Java

The source folder contains a Java 21 exercise runner that demonstrates the Core Java exercise topics.

Compile and run:

```bash
javac -d out src/com/cognizant/upskilling/corejava/CoreJavaExercises.java
java -cp out com.cognizant.upskilling.corejava.CoreJavaExercises
```

Additional folders contain JDBC, Java modules, and socket examples that are best run separately.
