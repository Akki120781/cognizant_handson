# Exercise 2: E-commerce Platform Search Function

This exercise compares linear search and binary search for product lookup.

## Complexity

| Algorithm | Best case | Average case | Worst case |
| --- | --- | --- | --- |
| Linear search | O(1) | O(n) | O(n) |
| Binary search | O(1) | O(log n) | O(log n) |

Binary search is better for frequent product lookups when the product list is already sorted by product ID. Linear search is still useful for small or unsorted lists.
