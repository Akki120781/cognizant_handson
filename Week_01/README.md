# Week 1 Hands-on

Week 1 covers the handbook's first 5 study days:

1. Engineering Concepts - Design Patterns and Data Structures/Algorithms
2. Programming Languages - PL/SQL, JUnit/Mockito, and SLF4J

## Verification notes

- Plain Java exercises are dependency-free and compile with `javac`.
- PL/SQL exercises are Oracle SQL scripts and should be run in an Oracle-compatible environment with `DBMS_OUTPUT` enabled.
- JUnit, Mockito, and SLF4J exercises include Maven `pom.xml` files because they require external dependencies.
