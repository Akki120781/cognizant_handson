# JPA, Hibernate, and Spring Data JPA

JPA is a specification that defines how Java objects are mapped to relational database tables.

Hibernate is an ORM framework and one implementation of the JPA specification. It provides the runtime behavior for entity persistence, sessions, transactions, and SQL generation.

Spring Data JPA is an abstraction over JPA providers such as Hibernate. It reduces repository boilerplate by generating CRUD and query-method implementations from repository interfaces.
