# Spring REST using Spring Boot 3

Implemented exercises:

- Create a Spring Web project using Maven
- Load Country from Spring XML configuration
- Hello World RESTful web service
- Country REST web service
- Get country based on country code
- Authentication service that returns JWT

Run from `spring-learn` with Maven:

```bash
mvn spring-boot:run
```

Useful endpoints:

- `GET /hello`
- `GET /country`
- `GET /countries`
- `GET /countries/{code}`
- `GET /authenticate` with HTTP Basic credentials `admin:pwd` or `user:pwd`
